# Advance Python Programming and Application,  NAVTTC Course 2026

A comprehensive Python programming course designed for beginners, covering fundamental concepts through object-oriented programming.

## 📚 Course Overview

This repository contains complete course materials including lecture notebooks, practice exercises, in-class tasks, and homework assignments. The course is structured to progressively build programming skills from basic syntax to advanced OOP concepts.

## 🎯 Learning Objectives

By completing this course, students will be able to:
- Write clean, efficient Python code
- Understand and apply fundamental programming concepts
- Work with various data structures (lists, tuples, sets, dictionaries)
- Create reusable functions and modules
- Apply object-oriented programming principles
- Build real-world applications

## 📋 Course Structure

### Week 1: Python Basics
- **Topic 1: Print Function**
  - Basic printing, parameters (sep, end)
  - Escape sequences and formatting
  - F-strings and string interpolation
  
- **Topic 2: Variables and Data Types**
  - Variable creation and naming conventions
  - Data types: int, float, str, bool
  - Type conversion and user input

### Week 2: Operations and Control Flow
- **Topic 3: Operators and Expressions**
  - Arithmetic, relational, and logical operators
  - Membership and identity operators
  - Operator precedence
  
- **Topic 4: Conditional Statements**
  - if, elif, else statements
  - Nested conditionals
  - Ternary operators

### Week 3: Loops and Iteration
- **Topic 5: Loops**
  - while loops
  - for loops and range()
  - Loop control (break, continue, pass)
  - Nested loops

### Week 4: Data Structures I
- **Topic 6: Lists**
  - Creating and accessing lists
  - List methods and operations
  - List comprehensions
  - Nested lists and algorithms

### Week 5: Data Structures II
- **Topic 7: Dictionaries**
  - Creating and modifying dictionaries
  - Dictionary methods
  - Nested dictionaries
  - Dictionary comprehensions

### Week 6: Data Structures III
- **Topic 8: Sets and Tuples**
  - Tuple creation, immutability, and unpacking
  - Set operations (union, intersection, difference)
  - Frozen sets
  - When to use each data structure

### Week 7: Functions
- **Topic 9: Functions**
  - Defining and calling functions
  - Parameters and return values
  - Default parameters and *args/**kwargs
  - Lambda functions
  - Scope (local vs global)

## 🚀 Getting Started

### Prerequisites
- Python 3.8 or higher
- Jupyter Notebook or JupyterLab
- VS Code
- Basic computer literacy

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/python-course.git
   cd python-course
   ```

2. **Install Python** (if not already installed)
   - Download from [python.org](https://www.python.org/downloads/)
   - Verify installation: `python --version`

3. **Install Jupyter Notebook**
   ```bash
   pip install jupyter notebook
   ```

4. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```
